A terminal client for Autorply. Most things you'd do in the agent dashboard work as commands you can script or pipe into other tools.

- **[Commands](/cli/commands)** — the full command reference: conversations, contacts, inboxes, agents, help centers.
- **[Scripting & output](/cli/scripting)** — JSON / CSV / quiet mode, shell completions, automation recipes.

## Install

Use the official installation method provided by your Autorply administrator.

## Log in

You'll be asked for:

- **Base URL** — your Autorply instance.
- **API Key** — from your agent profile.
- **Account ID** — the number in your dashboard URL.

Credentials are validated before saving.

## Shell completions

Tab-completion works for commands, subcommands, and flags.

## Help

Use the CLI help command to view available commands.

## Uninstalling

Remove the saved credentials and the local CLI binary from your device.
